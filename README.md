# 🥃 Scotland Whisky Map — Single Malts ~10 ans

Carte complète des whiskies single malt d'Écosse aux alentours de **10 ans d'âge**, organisée par région.

> Toutes les expressions listées ici ont entre **8 et 12 ans** de vieillissement.

---

## 🗺️ Les 6 Régions

| Région | Caractère général |
|--------|-------------------|
| [Speyside](#-speyside) | Fruité, floral, miel, vanille |
| [Highlands](#-highlands) | Diversifié, bruyère, fruits secs, tourbe légère |
| [Islay](#-islay) | Tourbé, fumé, iodé, sel marin |
| [Lowlands](#-lowlands) | Léger, floral, herbacé, accessible |
| [Campbeltown](#-campbeltown) | Sel, tourbe douce, fruits, complexe |
| [Islands](#-islands) | Maritime, tourbe variable, épicé |

## 📝 Notes

- Les prix sont indicatifs pour la France (grandes surfaces spécialisées / internet)
- **NAS** = No Age Statement (sans âge déclaré, souvent ~10 ans)
- **ppm** = parties par million (mesure du phénol/tourbe)
- **Cask Strength** = embouteillé à la force du fût, non dilué (~55-65% vol)
- La réglementation SWA impose **minimum 3 ans** de vieillissement en fût de chêne en Écosse
